   Widget for alarm handling

   Shows details on an alarm, e.g., identifying application, time of being
   issued, acknowledged, silenced, etc. Also shows alarm views when possible.

   :param config: pswamp config file.
   :param alarm_uuid: UUID of the alarm to be shown.
   :type alarm_uuid: uuid.uuid
   :param alarm_data: Data for the alarm to be shown.
   :type alarm_data: dict
   :param measurement_data_available: If this is False, alarm views will not show.
   :type measurement_data_available: bool
   :param grid_view: Allow content to be shown on the SLD.
   :type grid_view: GridViewContainer
