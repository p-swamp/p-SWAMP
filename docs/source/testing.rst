Test environments
=================

New appliations need to be tested rigorously, all the way from development of ideas towards online operating applications. 

This functionality is used in the examples, but also for unit testing.

-----------------------
Functionality for playing back stored PMU data (from real measurement devices, or synthetic data) is demonstrated in 
:doc:`one of the examples <examples/example_csv_playback>`.
